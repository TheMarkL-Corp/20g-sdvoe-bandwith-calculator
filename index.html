AVP-2000 Calculations — All values are theoretical average bandwidth. Peak is 5–6% higher. Dual-Link: SFP+ A+B bonded at 20 Gbps — Link A carries S0 + all aux up to 10G, Link B carries overflow + S1. Light compression (5:6:5 mapping, ratio 0.44–0.67) auto-activates in Single-Link when H Active ≥ 3000 px, fps ≥ 45 Hz, FRC off. Dual-Link is always fully uncompressed. S1 always outputs RGB 24bpp. Max input: 8192×4096. API + firmware required to switch Link Mode before routing in AV Manager.
